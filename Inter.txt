HTML (HyperText Markup Language) :----

	HTML ek markup language hai jo website ka structure banane ke liye use hoti hai. 
	Isse headings, paragraphs, images, forms, buttons aur links create karte hain.
	HTML website ki foundation hoti hai.

Example Answer:
		"HTML is a markup language used to create the structure of web pages. It defines elements such as headings, paragraphs, images, forms, and links."


-----------------------------------------------------------------------------------------------------------------------


CSS (Cascading Style Sheets) :----

	CSS website ko attractive banane ke liye use hoti hai.
	Isse colors, fonts, spacing, layouts aur responsiveness manage karte hain.
	HTML structure deta hai aur CSS styling.

Example Answer:
		"CSS is used to style web pages by controlling colors, layouts, fonts, spacing, and responsiveness."


-----------------------------------------------------------------------------------------------------------------------


JavaScript :-----

	JavaScript ek programming language hai.
	Website ko interactive banati hai.
	Buttons click karna, forms validate karna, API data fetch karna JavaScript se hota hai.

Example Answer:
		"JavaScript is a programming language used to add interactivity and dynamic behavior to websites."

React.js :------

	React ek JavaScript library hai.
